Child selectors

De child (kind) selector, zoals ook staat in de naam, zich alleen richt op kinderen.

De syntax voor de "direct child selector" vereist een simpele aanpassing. In plaats van een spatie tussen voorouder en afstammeling zoals in het geval van de descendant selector hebben we nu het groter dan teken (>) tussen ouder ("parent") en kind ("child"). Het > teken mag omgeven worden door één of meer spaties.

parent > child

Voorbeeld: div > h2 (in dit geval child selector zich richt alleen op h2 tag binnen de div tag en niet binnen de section. De kleur van de eerste h2 is rood, de tweede h2 blijft zwart)

CSS: div > h2{ color: red; }

Adjacent selectors

Adjacent sibling element-selectors kun je gebruiken om de stijl vast te leggen van een element op een vergelijkbaar niveau (dus geen kind), die direct op elkaar volgen. In de stijlregel wordt tussen de elementen een plusteken (+) geplaatst, naar wens voorafgegaan en gevolgd door één of meer spaties.

Voorbeeld: In de onderstaande code krijgt alleen h4 tag die buiten de section staat en direct na section volgt de achtergrondkleur

CSS: section + h4{ background-color: yellow; }

general selectors

he general sibling selector selects all elements that are next siblings of a specified element.

Voorbeeld: In de onderstaande code krijgen alle h5 tags die buiten de section staat de achtergrondkleur.

CSS: section ~ h5{ background-color: greenyellow; }
