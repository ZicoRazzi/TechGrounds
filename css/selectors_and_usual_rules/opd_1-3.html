Opdracht 1

Er zijn drie manieren om CSS in je HTML pagina's te krijgen:

CSS Stylesheet koppelen aan HTML pagina. De meest ideale oplossing is 1 of meer stylesheet(s) te maken die voor alle pagina's van de website gelden en deze aan de HTML-pagina te koppelen (linked).

CSS stijlregels opnemen in de HTML pagina. Bij uitzondering, als voorbeeld-pagina's of omdat het slechts 1 uitzondering bevat, kunnen specifiek voor 1 HTML-document geldende regels ook opgenomen worden in het document.

CSS stijlregels invoegen per HTML-element. De laatste en ook minst wenselijke opname van specifieke stijlregels is de inline methode. Hierbij worden per HTML-element zoals bijvoorbeeld deze <p> element de stijlregels opgegeven.

Opdracht 2

CSS syntax

De CSS-syntaxis bestaat uit 3 delen: een selector, een attribute en een waarde.

Syntaxis: selector {attribute: value}

De selector vertegenwoordigt het HTML-element dat we willen opmaken. Bijvoorbeeld:

.box{ width: 100px; height: 100px; border-radius: 10px; background-color: rgb(207, 76, 15) }

Opdracht 3

Verschillende selectors

Type selectors

De type selector komt overeen met de HTML elementen. Met andere woorden, het selecteert alle elementen van het gegeven type in een document.

Voorbeeld: alle h3 tags op deze pagina hebben een kleur.

h3{ color: #0b0; }

Descendant selectors

De descendant-combinator - meestal weergegeven door een enkele spatie combineert twee selectors zodat elementen die overeenkomen met de tweede selector worden geselecteerd als ze een voorouder hebben (ouder, ouder, ouder, ouder, ouder, enz.) element dat overeenkomt met de eerste selector. Selectors die gebruik maken van een descendant-combinator worden descendant selectors genoemd.

Voorbeeld: span tag in bovenstaande p tag

p span{ font-size: 22px; font-weight: bold; }

Class selectors

De class selector matcht elementen op basis van de inhoud van hun classattribuut. De naam van een class selector begint met een punt (.), maar bestaat verder uit underscores, hyphens, letters en cijfers, net als de naam van een element selector. Het grootste verschil is dat terwijl de naam van een element selector een tag in een HTML document aanduidt, de naam van een class selector vrij is. De naam wordt gekozen door de CSS ontwikkelaar.

Voorbeeld: de bovenstaande p tag met de class "class-selector"

.class-selector{ font-style: italic; font-family: Verdana, sans-serif; font-size: 20px; }
