Opdracht 1

Er zijn drie manieren om CSS in je HTML pagina's te krijgen:

CSS Stylesheet koppelen aan HTML pagina. De meest ideale oplossing is 1 of meer stylesheet(s) te maken die voor alle pagina's van de website gelden en deze aan de HTML-pagina te koppelen (linked).

CSS stijlregels opnemen in de HTML pagina. Bij uitzondering, als voorbeeld-pagina's of omdat het slechts 1 uitzondering bevat, kunnen specifiek voor 1 HTML-document geldende regels ook opgenomen worden in het document.

CSS stijlregels invoegen per HTML-element. De laatste en ook minst wenselijke opname van specifieke stijlregels is de inline methode. Hierbij worden per HTML-element zoals bijvoorbeeld deze <p> element de stijlregels opgegeven.
