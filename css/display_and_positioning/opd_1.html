Opdracht 1

Welke position values zijn er? Leg voor elke position values uit wat het doet. Geef ook aan welke default position waarde elementen hebben die je gebruikt in html.

Antwoord:

  • static: Standaardwaarde. Elementen worden weergegeven in volgorde, zoals ze verschijnen in de code.
  • absolute: Het element wordt gepositioneerd ten opzichte van zijn eerste gepositioneerde (niet statische) parent element
  • fixed: Het element is gepositioneerd ten opzichte van het browservenster
  • relative: Het element wordt gepositioneerd ten opzichte van zijn normale positie, dus "left:20px" voegt 20 pixels toe aan de LINKER positie van het element ten opzichte van het browservenster
  • sticky: Het element wordt gepositioneerd op basis van de scroll positie van de gebruiker
  • initial: Stelt deze eigenschap in op de standaardwaarde.
  • inherit: Erft deze eigenschap van zijn parent element.
