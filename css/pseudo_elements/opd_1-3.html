Opdracht 1

Kun je enkele voorbeelden geven van pseudo-elementen (minimaal 4)? Beschrijf met je eigen woorden wat deze pseudo-elementen inhouden.

Antwoord.

De meest voorkomende pseudo elementen:

  • ::after

    • creëert een pseudo-element dat de last child is van het geselecteerde element. Vaak gebruikt om cosmetische inhoud aan een element toe te voegen met behulp van de content eigenschap.

  • ::before

    • creëert een pseudo-element dat de first child is van het geselecteerde element. Vaak gebruikt om cosmetische inhoud aan een element toe te voegen met behulp van de content eigenschap.

  • ::first-line

    • past stijlen toe op de eerste regel van een blok-element. De lengte van de eerste regel is afhankelijk van vele factoren, waaronder de breedte van het element, de breedte van het document en de lettergrootte van de tekst.

  • ::first-letter

    • past de stijlen toe op de eerste letter van de eerste regel van het blok-element, maar alleen als er geen andere voorafgaande inhoud is (zoals afbeeldingen of inline tabellen).

Opdracht 2

Er zijn 14 verschillende pseudo elementen. Je hebt in de voorgaande opdracht reeds 4 kunnen benoemen. Kun je de andere 10 ook benoemen en aangeven wat ze precies doen?

Antwoord.

Andere tien pseudo elementen:

  • ::backdrop

    • is een box ter grootte van de viewport die wordt weergegeven direct onder elk element dat wordt gepresenteerd in volledig-scherm modus.

  • ::file-selector-button

    • vertegenwoordigt de knop van een input van het type="file".

  • ::grammar-error

    • vertegenwoordigt een tekstsegment dat door de user agent is gemarkeerd als grammaticaal onjuist.

  • ::marker

    • selecteert het markeringsvak van een li element, dat meestal een opsommingsteken of nummer bevat.

  • ::part

    • vertegenwoordigt elk element binnen een shadow tree dat een overeenkomend part attribuut heeft.

  • ::placeholder
    • vertegenwoordigt de placeholder-text in een input of textarea element.
  • ::selection

    • past stijlen toe op het deel van een document dat door de gebruiker is gemarkeerd (zoals klikken en slepen met de muis over tekst).

  • ::slotted()

    • vertegenwoordigt elk element dat in een slot is geplaatst in een HTML template

  • ::spelling-error

    • vertegenwoordigt een tekstsegment dat door de user agent is gemarkeerd als onjuist gespeld.

  • ::target-text

    • vertegenwoordigt de tekst waarnaar is gescrolld als de browser scroll-naar-tekst fragmenten ondersteunt. Hiermee kunnen auteurs kiezen hoe ze dat gedeelte van de tekst willen markeren.
