Arrays & Loops

Array

Wat is een array? Het is gewoon een lijst (een reeks) van waarden. In plaats van één enkele variabele om één waarde op te slaan, kan je een array variabele gebruiken voor het opslaan van een willekeurig aantal waarden. Die waarden vormen de elementen van de array.

  • een array is een data store
  • een array bevat geïndexeerd elementen
  • indexen starten vanaf nul en verhogen met één voor elk element in de array
  • voor toegang tot een element van een array, gebruik je de index tussen vierkante haken
  • een array kan elk soort datatype bevatten, met inbegrip van andere arrays

Scope

In JavaScript heb je een concept genaamd scope. Door middel van scope kan je goede code maken en errors beperken. Scope definieert de toegankelijkheid van een variable. Variable zijn onder andere de const, var, let maar ook functies en objects.

Er zijn twee soorten scope: de local en de global. Binnen Javascript zijn er twee verschillende type scope

Bij een globale scope, is de variable bereikbaar in het hele javascript document, binnen de functie en buiten de functie. De variable wordt dan buiten de code gedefinieerd. Bij een lokale scope is het alleen bereikbaar binnen de functie zelf.
